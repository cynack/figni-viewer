## 開発方法

1. `npm install`で必要なパッケージをダウンロード
2. src/のファイルを編集
3. `npm run build`で dist/に*figni-viewer.js*を出力

## 便利コマンド

- `npm run serve`
  - 検証用の HTML(./index.html)のサーバーを立ててくれるよ
- `npm run watch`
  - src/に変更があったら自動でビルド(development)
