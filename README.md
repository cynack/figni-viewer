## 開発方法

1. `pnpm install`で必要なパッケージをダウンロードする
2. src/のファイルを編集(index.html はテスト用の html ファイル)
3. `pnpm watch`で test/にテスト用のファイルを出力し、`pnpm test`でサーバーを起動して検証
4. `pnpm build`で dist/にビルド
